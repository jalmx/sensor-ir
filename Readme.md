# Sensor IR with LM358

Project designed with KiCAD

## Materials

- 1 LM358N
- 2 2N2222
- Led Infrared IR333
- Phototransistor PT331C
- Resistors 10k, 333k,5.6k, 1k
- 1 Potenciometer 10k
- 1 Relay 5VDC
